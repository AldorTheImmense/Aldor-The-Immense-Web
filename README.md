# Aldor The Immense - Web Version

This is a static HTML/CSS/JavaScript version of the original Windows Forms app.

## Files

- `index.html` - main page
- `style.css` - styling
- `script.js` - shop generator and random table logic
- `assets/` - icon and Aldor image

## How to run locally

Open `index.html` in a browser.

## How to access the Github Pages

Go to  `https://aldortheimmense.github.io/Aldor-The-Immense-Web/`

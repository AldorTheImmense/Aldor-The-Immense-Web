# Aldor The Immense - Web Version

This is a static HTML/CSS/JavaScript version of the original Windows Forms app.

## Files

- `index.html` - main page
- `style.css` - styling
- `script.js` - shop generator and random table logic
- `assets/` - icon and Aldor image

## Save codes

Use **Copy Save Code** to copy the current shop inventory and custom item lists. Use **Load Save Code** on another browser or computer to restore them.
